# DeltaFStation 快速启动指南

## 🚀 5分钟快速开始

### 1. 环境准备
确保你的系统已安装 Python 3.8+：
```bash
python3 --version
```

### 2. 启动系统
```bash
# 方法1: 使用启动脚本（推荐）
./start.sh

# 方法2: 直接运行
python3 run.py
```

### 3. 访问系统
打开浏览器访问: http://localhost:5000

## 📊 快速体验

### 步骤1: 上传数据
1. 点击"上传数据"按钮
2. 选择CSV文件（或使用示例数据）
3. 确认上传成功

### 步骤2: 创建策略
1. 点击"创建策略"按钮
2. 填写策略信息：
   - 名称: 移动平均策略
   - 类型: 技术分析
   - 描述: 基于5日和20日移动平均线
3. 点击"创建"

### 步骤3: 运行回测
1. 选择刚创建的策略
2. 选择数据文件
3. 设置回测参数：
   - 开始日期: 2023-01-01
   - 结束日期: 2023-12-31
   - 初始资金: 100000
4. 点击"运行回测"
5. 查看回测结果和图表

### 步骤4: 启动仿真
1. 点击"启动仿真"按钮
2. 选择策略和初始资金
3. 监控仿真状态
4. 查看交易记录

## 🔧 常用操作

### 数据管理
- **上传CSV**: 支持包含Date, Open, High, Low, Close, Volume列的文件
- **下载数据**: 从Yahoo Finance下载实时数据
- **数据预览**: 查看数据文件内容

### 策略管理
- **创建策略**: 定义交易规则和参数
- **编辑策略**: 修改策略配置
- **删除策略**: 移除不需要的策略

### 回测分析
- **运行回测**: 基于历史数据验证策略
- **查看结果**: 分析收益率、夏普比率等指标
- **图表展示**: 可视化回测结果

### 仿真交易
- **启动仿真**: 开始实时仿真交易
- **手动交易**: 执行手动买卖操作
- **监控状态**: 实时查看持仓和交易记录

## ⚠️ 注意事项

1. **数据格式**: CSV文件必须包含Date, Open, High, Low, Close, Volume列
2. **日期格式**: 日期格式应为YYYY-MM-DD
3. **数值格式**: 价格和成交量应为数值类型
4. **文件大小**: 建议单个文件不超过16MB

## 🐛 常见问题

### Q: 启动失败怎么办？
A: 检查Python版本和依赖包安装：
```bash
python3 --version
pip3 install -r requirements.txt
```

### Q: 数据上传失败？
A: 检查CSV文件格式，确保包含必需的列

### Q: 回测没有结果？
A: 检查数据文件是否包含指定日期范围的数据

### Q: 仿真无法启动？
A: 确保已创建策略并选择了有效的数据文件
